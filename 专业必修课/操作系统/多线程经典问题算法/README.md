# 多线程经典问题算法

本文件夹包含多线程经典问题的算法实现伪代码及相关配置文件。

[.editorconfig](.editorconfig) 文件用于定义代码风格和格式化规则，确保代码的一致性和可读性。

[common.h](common.h) 文件包含编写伪代码时使用的`TRUE`变量、信号量的`down`和`up`操作。

[生产者消费者问题.c](生产者消费者问题.c)、[读者写者问题.c](读者写者问题.c)、[哲学家就餐问题.c](哲学家就餐问题.c)、[睡眠的理发师问题.c](睡眠的理发师问题.c)文件分别包含对应经典多线程问题的算法实现伪代码。这些伪代码展示了如何使用信号量来解决多线程同步和互斥的问题。开头引入`common.h`和声明一些函数，避免某些静态分析工具报错，而导致可读性下降。
